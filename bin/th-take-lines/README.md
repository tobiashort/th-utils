```
Usage:
  th-take-lines [OPTIONS] [File]

Options:
  -H, --head <Head>    Number of beginning lines to take
  -t, --tail <Tail>    Number of ending lines to take
  -l, --lines <Lines>  Specific lines to take, e.g. 2,3,5-10,24
  -h, --help           Show this help message and exit

Positional arguments:
  File                 The file to read. Reads from Stdin if not specified.

```
