```
Usage:
  url-encode [OPTIONS] [String]

Options:
  -p, --path   Escapes the string so it can be safely placed inside a URL path segment
  -q, --query  Escapes the string so it can be safely placed inside a URL query
  -a, --all    Escape all characters
  -h, --help   Show this help message and exit

Positional arguments:
  String       The string to encode. Reads from Stdin if not specified.

```
